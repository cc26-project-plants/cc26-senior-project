## Getting Started: Frontend  
1. move to the frontend directory  
```bash
cd frontend
```
2. download dependencies  
```bash
npm i
```
3. run the development server  
```bash
npm run dev
# or
yarn dev
```
4. open [http://localhost:3000](http://localhost:3000) with your browser to see the app
